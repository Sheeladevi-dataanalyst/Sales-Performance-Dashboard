# ShopNest Store Sales & Customer Insights Dashboard
## Project Objective
To design an interactive Power BI dashboard that transforms raw retail data into meaningful insights, enabling stakeholders to monitor KPIs, optimize sales strategies, improve customer experience, and enhance overall business performance.

## PROCESS
### Data Collection
Gathered raw data from Excel and SQL sources, including sales, customers, products, and orders.
### Data Cleaning
Removed duplicates, handled missing values, and standardized data formats to ensure accuracy.
### Data Transformation
Used Power Query to clean, reshape, and create new columns such as revenue and delivery status.
### Data Integration
Merged and combined multiple datasets into a unified structure for analysis.
### Data Modeling
Built relationships between tables and designed a star schema for efficient data analysis.
### DAX Calculations
Created measures like total sales, average rating, delayed orders, and other KPIs.
### Dashboard Creation
Developed an interactive Power BI dashboard with charts, KPI cards, filters, and top-N analysis.
### Insights & Reporting
Analyzed trends, identified key business insights, and presented findings through visuals.
